
## 1. 设置pip3的源为清华源
    pip3 config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
    执行这条命令会生成pip3的配置文件
    ~/.config/pip/pip.conf
    pip3 config get global.index-url



## 其它命令
sudo apt install python3-pip    安装pip
pip3 --version 查看版本
